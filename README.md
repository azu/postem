# Postem

TomblooとかTombfixみたいなElectronアプリ。

## Feature

- Twitter、はてなブックマークとかへのクロスポスト
- [textlint](https://github.com/textlint/textlint "textlint")でのリアルタイムLint

![screenshot](https://monosnap.com/file/9WtShAGiCilmCOLtuGEWnfphfpKndf.png)

## Installation

    npm install

## Usage

    npm start
    # develop

### production

    npm run build
    ./bin/cmd.js
    # production
